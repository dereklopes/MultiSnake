# MultiSnake

A twist on the classic game Snake - allows multiple players to connect and play over a network.

## Prerequisites

You need pygame and python3 installed on your system. You can install pygame with pip:
```
pip install pygame
```

## Running

On the server, run:
```
python snake.py
```
On the client(s), run:
```
python client.py
```
